# Projeto: Moderador de Mensagens em Redes Sociais

Este projeto é um **moderador de mensagens de redes sociais**, desenvolvido com o uso de **LLM**.  
Para isso, foram escolhidos os modelos:

- **GPT-4o mini**
- **Omni-moderation-latest**

Ambos desenvolvidos pela **OpenAI**.

---

## Como utilizar

Para utilizar o moderador, basta preencher as seguintes variáveis:

- `message` → conteúdo da mensagem  
- `date` → data de envio  
- `user` → usuário que enviou a mensagem  

Ao final, é produzido um **relatório da mensagem**.
